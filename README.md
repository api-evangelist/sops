# SOPS ()
Secrets OPerationS - A tool for managing secrets in configuration files by encrypting values while leaving keys unencrypted, supporting multiple key management services like AWS KMS, GCP KMS, Azure Key Vault, and PGP.

**URL:** [Visit APIs.json URL](https://github.com/mozilla/sops)
